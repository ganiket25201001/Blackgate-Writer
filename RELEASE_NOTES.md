## 📥 Installation

### Windows
1. Download `Blackgate.Writer_0.1.0_x64_en-US.msi`
2. Double-click to run installer
3. Follow setup wizard
4. App launches automatically

### macOS
1. Download `Blackgate.Writer_0.1.0_x64.dmg`
2. Double-click to mount disk image
3. Drag app to Applications folder
4. Launch from Applications or Spotlight

### Linux
1. Download `blackgate_writer-0.1.0-x86_64.AppImage`
2. Make executable: `chmod +x blackgate_writer-0.1.0-x86_64.AppImage`
3. Run directly: `./blackgate_writer-0.1.0-x86_64.AppImage`
4. Or install system-wide (see BUILD_GUIDE.md)

## 🎮 Quick Start

### Creating Your First Document
1. **Launch** the app
2. **File → New** or press **Ctrl+N**
3. **Type** your content
4. **Format** using ribbon buttons
5. **Save** with **Ctrl+S**

### Using Templates
1. **File → New from Template...**
2. Choose template (Business Letter, Report, etc.)
3. **Edit** the template content
4. **Save** with **Ctrl+S**

### Opening Recent Files
1. Click **File** menu
2. Scroll to "Recent Files" section
3. Click file to open
